# NEXUS Gaming — Landing Page Template

Dark theme gaming landing page template built with Next.js 16 and Tailwind CSS v4. Inspired by [kodus.io](https://kodus.io/) design language.

## Preview

- Pixel font headings (Press Start 2P)
- Dark grid background with space invader decorations
- Gold/amber accent gradient
- Glassmorphism cards
- Smooth scroll animations
- Interactive chat widget
- Fully responsive (mobile-first)

## Tech Stack

- **Next.js 16** (App Router, Turbopack)
- **Tailwind CSS v4**
- **TypeScript**
- Zero external animation libraries (CSS-only animations)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
